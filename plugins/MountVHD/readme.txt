Mounts a VHD disk stored in Azure Blob Storage as a drive on a Web or Worker Role.

Settings used by this plugin:

Two10.MountVHD.DriveLetter : 
  The mount point (drive letter) for the disk 
  (i.e. X)

Two10.MountVHD.StorageAccount : 
  The storage connection string containing the VHD blob

Two10.MountVHD.BlobUri :
  The location of the VHD file in blob storage 
  (i.e. container/file.vhd)
